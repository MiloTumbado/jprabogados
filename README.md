# JPR Abogados - Corporate Website

Welcome to the new digital home of **JPR Abogados**. This project is a modern, high-performance web application designed to reflect the firm's professionalism and expertise in Labor Law.

## Project Overview

This website has been rebuilt from the ground up using modern web technologies to ensure speed, security, and a premium user experience. It features a clean, corporate aesthetic that aligns with the firm's brand identity.

### Key Features

*   **Premium Design:** A bespoke UI built with `styled-components` using a sophisticated color palette (Navy Blue & Gold) and typography (Playfair Display & Lato).
*   **Smooth Animations:** Subtle entrance animations and transitions powered by `Framer Motion` to enhance engagement without compromising professionalism.
*   **Responsive Layout:** Fully optimized for all devices, from desktop monitors to mobile phones.
*   **Contact Integration:** A functional contact form that connects to a Netlify backend (ready for configuration).
*   **Fast Performance:** Powered by Vite, ensuring instant page loads and smooth navigation.

## Technology Stack

*   **Frontend Library:** React (v18)
*   **Build Tool:** Vite
*   **Styling:** Styled Components
*   **Routing:** React Router DOM
*   **Animations:** Framer Motion
*   **Hosting:** Netlify (Ready)

## Getting Started (For Developers/Maintenance)

If you need to make updates or run the project locally, follow these steps:

### Prerequisites

*   Node.js (v16 or higher)
*   npm (Node Package Manager)

### Installation

1.  Clone the repository or unzip the project file.
2.  Open your terminal in the project folder.
3.  Install dependencies:
    ```bash
    npm install
    ```

### Local Development

To view the site on your local machine:
```bash
npm run dev
```
Open your browser to the URL shown (usually `http://localhost:5173`).

### Building for Production

To create the final files for deployment:
```bash
npm run build
```
The output will be in the `dist/` folder.

## Deployment

This project is configured for seamless deployment on **Netlify**.
1.  Connect your repository to Netlify.
2.  The `netlify.toml` file included in the project will automatically handle the build settings.
